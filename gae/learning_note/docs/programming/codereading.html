如何阅读源代码¶

注解

本章节内容主要是针对 code reading-The Open Source Perspective 这本书的学习笔记, 下文中相应的页码指的是书内对应的页码.

为什么要阅读源代码¶

阅读源代码的原因主要包括:

  1. code as literature 享受
  2. code as exemplar 标本,模范
  3. maintainance 维护
  4. evolution 更新
  5. reuse 重用
  6. inspections 审查
