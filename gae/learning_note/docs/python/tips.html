mixin¶

注解

mixin 是一种常用的OO编程方法 支持 mixin 的语言通常需要满足:

  1. 多重继承
  2. 动态绑定
  3. 运行时更改类

# coding:utf-8
#!/usr/bin/env python

class A:
    pass

class B:

    def p(self, value):
        print value

class C(A):
    pass

if __name__ == "__main__":
    a = C()
    #a.p()   # will get error
    C.__bases__ += (B,) # dynamically change the inheriting tree
    a.p("hello")
