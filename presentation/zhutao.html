什么是Python

Python 是一种 通用 ( general purpose) 语言, 它已经在各种领域都有 了很广泛的应用.譬如:

  • 大型的软件构建( openerp, Zope 等)
  • Web框架 (Django, Web.py, 等)
  • 自然语言处理( NLTK )
  • 科学计算 (NumPy)
  • 日常的脚本使用 (写写脚本, 分析日志等)
  • 等等

Python 目前是 Google 的第二大开发语言.

关于Django

Django 是一种高效, 简洁, 适合快速开发的web框架.

它有如下的特点:

  1. MTV(Model, Template, View, 和MVC对应)
  2. ORM(独立于特定数据库, 高层封装)
  3. url dispatch(使用与此正则表达式来进行url映射)
  4. 集成的应用(如admin, auth等)
