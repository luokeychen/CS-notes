定义
========================================

如果一个树满足下面的特征:

#. 每个节点具有唯一的值
#. 左子树和右子树也是二叉搜索树
#. 一个节点的左子树仅包含值比此节点小的节点
#. 一个节点的右子树仅包含值比此节点大的节点
